Free Online Courses

For R:

  • R for Data Science (2e) - Comprehensive free book by Hadley Wickham
  • RStudio Primers - Interactive tutorials
  • Swirl - Learn R programming interactively

For Python:

  • Python for Everybody - Free course by University of Michigan
  • Kaggle Learn - Practical micro-courses
  • DataCamp - First course free, student discounts available

Statistics & Machine Learning:

  • StatQuest - Excellent YouTube channel for statistics
  • Introduction to Statistical Learning - Free book with R and Python code
  • fast.ai - Practical deep learning for coders

University Resources

At Leeds:

  • Check for data science modules in your program
  • Attend research seminars and workshops
  • Join data science student groups
  • Explore Leeds Institute for Data Analytics (LIDA)

Advanced Topics to Explore

Once you‚Äôre comfortable with basics:

Data Science Skills

  • Version Control: Git and GitHub for collaboration
  • Reproducible Research: R Markdown, Quarto, Jupyter notebooks
  • Databases: SQL for working with large datasets
  • Big Data: Spark, cloud computing (AWS, Google Cloud)

Specialized Topics

  • Spatial Data Science: Working with geographic data (check out geocompx.org)
  • Machine Learning: Supervised and unsupervised learning
  • Deep Learning: Neural networks for complex patterns
  • Time Series Analysis: For temporal data
  • Network Analysis: For transport networks and relationships

Transport-Specific Skills

  • Transport Modeling: Learn about the A/B Street project
  • Route Planning: OpenTripPlanner, OSRM
  • GIS: QGIS, ArcGIS for spatial analysis
  • Transport Data: Working with GTFS, OpenStreetMap
