Practical: Basics

Welcome to the fundamentals! This section covers the essential concepts and tools you need to get started with data science.

Integrated Development Environments (IDEs) or the software you will use…

RStudio

RStudio has four main panels:

  1. Source Editor (top-left): Where you write and edit your R code
  2. Console (bottom-left): Where code is executed and results appear
  3. Environment/History (top-right): Shows your variables and command history
  4. Files/Plots/Packages/Help (bottom-right): View files, plots, manage packages, and access help

Tip: You can customize the layout by going to View → Panes → Pane Layout

VS Code

When working with Python and R in VS Code:

  1. Editor (left): Write your code
  2. Explorer (left sidebar): Navigate your project files
  3. Terminal (bottom): Run commands and see output
  4. Extensions Panel: Install packages and extensions for Python, R, and Quarto support

Popular Extensions: Python, R, Pylance, Quarto

Data Types and Structures

Data Types in R

R has several fundamental data types:

# Numeric (default for numbers)
x <- 42
x <- 3.14

# Integer (explicitly specified)
y <- 42L

# Character (text)
name <- "Alice"
sentence <- "Data science is fun!"

# Logical (boolean)
is_true <- TRUE
is_false <- FALSE

# Check the type
class(x)
typeof(x)

Key Data Types: - numeric: Real numbers (integers and decimals) - integer: Whole numbers only - character: Text strings - logical: TRUE or FALSE - complex: Complex numbers (rarely used in data science)

Data Structures in R

Vectors

A vector is the most basic data structure in R. It’s a sequence of values of the same type:

# Numeric vector
numbers <- c(1, 2, 3, 4, 5)

# Character vector
names <- c("Alice", "Bob", "Charlie")

# Logical vector
flags <- c(TRUE, FALSE, TRUE)

# Create sequences
seq_1_to_10 <- 1:10
seq_by_2 <- seq(0, 20, by = 2)

# Access elements
first_number <- numbers[1]
first_three <- numbers[1:3]

Data Frames

A data frame is like a table with rows and columns. Each column is a vector and can contain different data types:

# Create a data frame
students <- data.frame(
  name = c("Alice", "Bob", "Charlie"),
  age = c(20, 21, 19),
  grade = c("A", "B", "A")
)

# Access columns
students$name
students[, "age"]

# Access rows
students[1, ]

# View the data frame
head(students)
str(students)

Lists

A list is a flexible container that can hold elements of different types and structures:

# Create a list
my_list <- list(
  name = "Alice",
  age = 20,
  scores = c(85, 90, 88),
  data = data.frame(x = 1:3, y = 4:6)
)

# Access elements
my_list$name
my_list[[1]]
my_list[["scores"]]

Data Types in Python

Python has similar data types:

# Integer
x = 42

# Float (decimal numbers)
y = 3.14

# String (text)
name = "Alice"
sentence = "Data science is fun!"

# Boolean
is_true = True
is_false = False

# Check the type
type(x)
type(name)

Key Data Types: - int: Integers - float: Decimal numbers - str: Text strings - bool: True or False

Data Structures in Python

Lists

A list in Python is an ordered collection of items (can be mixed types):

# Create lists
numbers = [1, 2, 3, 4, 5]
names = ["Alice", "Bob", "Charlie"]
mixed = [1, "Alice", 3.14, True]

# Access elements (Python uses 0-based indexing)
first_number = numbers[0]
first_three = numbers[0:3]

# Modify lists
numbers.append(6)
names[0] = "Alicia"

Dictionaries

A dictionary is like Python’s equivalent to R’s named list. It stores key-value pairs:

# Create a dictionary
student = {
    "name": "Alice",
    "age": 20,
    "grade": "A"
}

# Access values
student["name"]
student.get("age")

# Modify values
student["grade"] = "A+"

NumPy Arrays

NumPy arrays are similar to R vectors - they hold multiple values of the same type:

import numpy as np

# Create arrays
numbers = np.array([1, 2, 3, 4, 5])
matrix = np.array([[1, 2, 3], [4, 5, 6]])

# Access elements
first = numbers[0]
first_three = numbers[0:3]

Pandas DataFrames

Pandas DataFrames are like R’s data frames - tables with rows and columns:

import pandas as pd

# Create a DataFrame
students = pd.DataFrame({
    "name": ["Alice", "Bob", "Charlie"],
    "age": [20, 21, 19],
    "grade": ["A", "B", "A"]
})

# Access columns
students["name"]
students.age

# Access rows
students.iloc[0]
students.loc[0]

# View the data
students.head()
students.info()

Comparison: R vs Python Data Structures

R Python Purpose
Vector List Sequence of values
Data Frame Pandas DataFrame Tabular data
List Dictionary or List Flexible container
Matrix NumPy Array Multi-dimensional data

Using R as a Calculator

One of the simplest ways to start with R is to use it like a calculator:

# Basic arithmetic
2 + 3
10 - 4
5 * 6
20 / 4
2 ^ 3  # exponentiation
10 %% 3  # modulo (remainder)

# Order of operations
(2 + 3) * 4

# Using variables
a <- 10
b <- 20
a + b
a * b

Common R Operators: - +: Addition - -: Subtraction - *: Multiplication - /: Division - ^: Exponentiation - %%: Modulo (remainder)

For a comprehensive guide on R operators, see: R Operators Reference

Understanding Packages

A package in R is a collection of functions, data, and documentation bundled together to extend R’s capabilities.

Installing and Loading Packages

# Install a package (do this once)
install.packages("tidyverse")

# Load a package (do this each session)
library(tidyverse)

Example: The tidyverse Package

tidyverse is a collection of packages for data manipulation and visualization:

# Once installed and loaded:
library(tidyverse)

# You now have access to functions like:
# - filter(): subset rows
# - select(): choose columns
# - mutate(): create new variables
# - ggplot(): create visualizations

Where to Find Package Information

  • CRAN Task Views: https://cran.r-project.org/web/views/
  • Package Documentation: Visit https://cran.r-project.org/ and search for the package
  • RStudio Package Guide: Type ?package_name in the console or search the Help tab
  • GitHub Repositories: Many packages have repos with vignettes and examples

Packages in Python

Python packages work similarly to R packages:

# Install a package (usually from the terminal)
# pip install pandas

# Import a package
import pandas as pd
import numpy as np

# Import specific functions
from matplotlib import pyplot as plt

Popular Python Data Science Packages: - pandas: Data manipulation - numpy: Numerical computing - matplotlib: Data visualization - seaborn: Statistical visualization - scikit-learn: Machine learning
